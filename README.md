# 📦 Frontend - Projeto Integrador (PI)

## 🚀 Tecnologias utilizadas
- React / Next.js  
- TypeScript  
- Tailwind CSS / CSS  
- Docker  

## Docker

Este projeto utiliza Docker para facilitar a execução em qualquer ambiente.

---

## ⚙️ Comandos Docker

### 🔹 1. Build da imagem

```bash
docker build -t frontendnotagest .
````

### 🔹 2. Rodar o container

```bash
docker run -d -p 4000:4000 frontendnotagest
````
